Why Woz

AI engineers do tasks.
Woz runs the show.

Every AI coding agent follows the same playbook. Woz operates a level above.

Tickets Projects

Leads entire projects

Creates Linear projects, breaks them into milestones, scopes every ticket, and automatically picks up unblocked work. Woz doesn't wait to be told what to do.

Planning Design docs

Writes design docs

Produces full technical design documents with architecture decisions, trade-offs, and scope. Plans entire projects, not just the next ticket.

Testing Screenshots

Tests with visual evidence

Tests changes end-to-end and provides screenshot evidence. You see exactly what changed before you merge. Trust, but verify.

PRs Triage

Triages all reviewers

Waits for your favorite code reviewers — Claude, Codex, humans — and triages feedback automatically. Pushes fixes, resolves threads, re-requests review.

Architecture

How it works

Woz is an OpenClaw fork, extended with a web server that connects to Linear and GitHub via webhooks.

1

Install with one command

Download Woz and connect your Linear, GitHub, and Slack accounts.

2

Woz listens via webhooks

A lightweight web server receives events from Linear and GitHub in real time.

3

Woz takes action

Plans projects, picks up tickets, writes code, tests with screenshots, triages reviews — autonomously.
