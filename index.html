What Woz can do

Your AI CTO handles the work that slows teams down.

Design docs & sprint planning

Writes technical design documents and breaks work into well-scoped tickets, ready for your team to ship.

Leads Linear projects

Monitors project boards and automatically picks up unblocked tickets. Keeps work moving without you nudging it.

Addresses code reviews

Automatically responds to GitHub PR review comments, pushes fixes, and resolves conversations.

Responds to mentions

Tag @woz in Linear or GitHub and it jumps in — answers questions, investigates issues, takes action.

How it works

Woz is an OpenClaw fork, extended with a web server that connects to Linear and GitHub via webhooks.

1

Install with one command

Download Woz and connect your Linear, GitHub, and Slack accounts.

2

Woz listens via webhooks

A lightweight web server receives events from Linear and GitHub in real time.

3

Woz takes action

Plans sprints, picks up tickets, writes code, addresses reviews — autonomously.
